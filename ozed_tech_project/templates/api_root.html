Inventory Items

  • GET /api/inventory/items/ - List all items
  • POST /api/inventory/items/ - Create new item
  • GET /api/inventory/items/{id}/ - Get item details
  • PUT /api/inventory/items/{id}/ - Update item
  • DELETE /api/inventory/items/{id}/ - Delete item
  • GET /api/inventory/items/low_stock/ - Get low stock items
  • GET /api/inventory/items/out_of_stock/ - Get out of stock items
  • POST /api/inventory/items/{id}/adjust_stock/ - Adjust stock quantity

Purchase Orders

  • GET /api/inventory/purchase-orders/ - List all orders
  • POST /api/inventory/purchase-orders/ - Create new order
  • GET /api/inventory/purchase-orders/{id}/ - Get order details
  • POST /api/inventory/purchase-orders/{id}/add_item/ - Add item to order
  • POST /api/inventory/purchase-orders/{id}/change_status/ - Change order status
  • POST /api/inventory/purchase-orders/{id}/receive_order/ - Mark as received & update inventory

  • POST /api/crm/customers/ - Create new customer
  • GET /api/crm/customers/{id}/ - Get customer details
  • GET /api/crm/customers/{id}/contacts/ - Get customer contacts
  • GET /api/crm/customers/{id}/interactions/ - Get customer interactions
  • GET /api/crm/customers/{id}/purchase_orders/ - Get customer orders
